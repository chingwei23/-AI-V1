▍五大主流 AI 助手總評比

了解各種工具的強項與極限，才能在對的情境用對的武器。

ChatGPT
(OpenAI)

月費 $20
🎯 最佳情境：寫程式、複雜邏輯推演、日常萬用助理。
👍 優勢：
  • 綜合能力最強的全能王者。
  • 邏輯推理與解題能力極佳。
  • 擁有強大的 GPTs 擴充外掛生態系。
👎 劣勢：
  • 中文寫作容易有「翻譯腔」。
  • 語氣有時過於死板、充滿機器感。

Claude
(Anthropic)

月費 $20
🎯 最佳情境：寫感人文案、閱讀數百頁長篇報表、企劃書潤飾。
👍 優勢：
  • 中文語氣最自然，最懂人話。
  • 極長文本的記憶力與處理能力無敵。
  • 回答極富人性，文案寫作首選。
👎 劣勢：
  • 免費版額度非常少。
  • 無法直接上網搜尋最新即時資訊。

Gemini
(Google)

月費 $20
🎯 最佳情境：快速搜尋即時新聞、整合 Google 雲端文件。
👍 優勢：
  • 與 Google 生態系完美整合 (Docs/Drive)。
  • 搜尋網路即時資訊速度極快。
  • 免費版速度就非常優秀。
👎 劣勢：
  • 回答品質有時不穩定。
  • 偶爾會出現邏輯錯誤的「幻覺」。

NotebookLM
(Google)

免費
🎯 最佳情境：內部知識庫建置、會議紀錄整理、法規與講義查詢。
👍 優勢：
  • 絕對不瞎掰！只根據你給的資料回答。
  • 能將枯燥文件轉成雙人對談的 Podcast 音訊。
👎 劣勢：
  • 不能當聊天機器人閒聊。
  • 完全無法回答資料庫以外的問題。

Manus
自主代理 Agent (最新趨勢)

邀請內測中
🎯 最佳情境： 全自動完成多步驟任務（例如：去網路上搜集十家競品資料，整理成 Excel 並發送郵件）。
👍 優勢：
  • 不只是對話，它能「自己操作瀏覽器與軟體」。
  • 丟給它一個大目標，它會自己拆解步驟並執行。
👎 劣勢：
  • 目前極難取得帳號。
  • 執行複雜任務時容易陷入死循環。

▍第一部分：黃金五指框架

與 AI 溝通的核心心法，寫出精準 Prompt 的必備五元素。

💡 景崴講師的進階筆記

  • 1. 限制是靈魂不要只告訴 AI 要做什麼，在 Prompt 裡面一定要讓 AI 清楚知道「絕對不能做什麼」。
  • 2. 保持批判性AI 是基於「已知」進行「未知」的預測。對 AI 產出的內容必須保持批判性，不要輕易相信，你是最終裁決者。
  • 3. 自我審查迴圈把 AI 產出的東西，再餵回去給它問：「請自評這是最好的選擇或答案嗎？如果是，為什麼？如果不是，請優化。」
  • 4. 逆向工程思維計畫生成後，請補上關鍵指令：「請給我 5 個此計畫可能會失敗的原因，並提供對策。」

▍第五部分：將系統部署上線

🌐 1. GitHub 免費網頁部署

  1. 前往 GitHub 官網 註冊並登入帳號。
  2. 點擊 New Repository 建立新專案。
  3. ⚠️
    極度重要：設定權限時，務必選擇「Public (公開)」！
    如果設為 Private，GitHub Pages (網頁託管服務) 將無法為你啟動伺服器。
  4. 上傳剛剛寫好的 index.html 檔案並 Commit changes 存檔。
  5. 前往專案的 Settings > Pages，將 Source 選擇為 `main` 分支並點擊 Save。等候 1-2 分鐘，專屬網址就會產出囉！

🔑 2. 為網頁注入靈魂：API 與 API Key

Q: 什麼是 API？

全名：Application Programming Interface (應用程式介面)

💡 秒懂比喻：去餐廳點菜的「服務生」。
你不需要知道廚房（AI 大模型）怎麼生火做菜的。你只要把點餐單（使用者的 Prompt）交給服務生（API），服務生就會走進廚房，並把做好的頂級料理（生成的文案）端回你的餐桌（你的靜態 HTML 網頁）。

👉 意義：讓你的普通網頁，瞬間擁有呼叫全世界最強 AI 算力的能力！

Q: 什麼是 API Key？

全名：API 專屬金鑰

💡 秒懂比喻：進高級餐廳的「專屬信用卡」。
服務生要幫你點餐前，必須確認「是誰在點餐？要扣誰的錢？」。API Key 就是一串亂碼密碼，用來驗證你的身份與計算使用額度 (Token)。

⚠️ 警告：絕對不能把 API Key 公開貼在網路上！這就像把信用卡密碼發在臉書上一樣，會被別人盜刷 AI 算力！

🖼️ 3. 網站放圖片的兩種法門

圖片載入方法 具體做法 優缺點比較
📂 方法一：相對路徑
(與程式碼同居)		 把圖片上傳到 GitHub，和 index.html 放在同一個專案資料夾裡。
程式碼寫：src="./logo.png"		 ✅ 優點：最穩定，圖片永遠跟著網頁走。 ❌ 缺點：更新圖片時需要手動上傳檔案。
🔗 方法二：絕對路徑
(雲端圖床網址)		 把圖片上傳到圖床(如 Imgur、Google Drive)，直接貼上對外公開的網址。
程式碼寫：src="https://..."		 ✅ 優點：程式碼乾淨，隨時換網址就能換圖。 ❌ 缺點：若圖床網址失效，你的網站就會破圖。
